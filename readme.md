# Keten: A simple Blockchain library

![Status](https://img.shields.io/badge/status-development-green) ![Language](https://img.shields.io/badge/language-C%2B%2B23-blue)

**Keten** is a blockchain implementation written to learn about the inner workings of blockchain technology. The implementation is far from perfect and is not the goal
of this project. To implement the blockchain I am using a set of crypto libraries for hashing ([`PicoSHA2`](https://github.com/okdshin/PicoSHA2) & [`monochypher`](https://github.com/LoupVaillant/monocypher)).
To implement the networking I am going to use my own network library I've written in C a while ago called [`Minimal Sockets`](https://github.com/WoutervanDixhoorn/Minimal-Sockets).

## Goals
1. [ ] Implement core Crypto logic.
2. [ ] Implement P2P Networking using [`Minimal Sockets`](https://github.com/WoutervanDixhoorn/Minimal-Sockets).
3. [ ] Use the blockchain in some other project/game to facilitate as somekind of currency system.
4. [ ] To be found during development....
